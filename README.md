# DeadZone BuildExt

A simple GitHub Actions launcher for DeadZone ROM builds.

## DeadZone Lite

Open **Actions → DeadZone Lite Xiaomi Build → Run workflow**.

The Lite workflow asks for only two values:

1. **Your name** — the builder name shown in build notifications.
2. **Direct URL of the Xiaomi ROM** — an HTTP or HTTPS link to the source ROM.

The launcher checks out the latest `main` branch of:

```text
hypermezo4-create/DeadZone_xiaomi-Lite
```

It then runs the normal DeadZone Lite process:

```text
Download → Extract → Build → Pack → Upload
```

The final ROM name is generated by the Lite engine from the detected ROM metadata, for example:

```text
DeadZone_ZIRCON_OS3.0.303.0.WNOCNXM_ChinaStable_V1.2.zip
```

## Required secrets

```text
PRIVATE_REPO_TOKEN or GH_TOKEN
GH_REPO
RCLONE_TOKEN_PATH
TELEGRAM_BOT_TOKEN
TELEGRAM_CHANNEL_ID
```

`PRIVATE_REPO_TOKEN` or `GH_TOKEN` must be able to read the private Lite engine and the repository containing the upload configuration.

GamingPlus and FrameworkPatcher workflows are separate and are not changed by the simplified Lite workflow.
